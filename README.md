Шифратор/Дешифратор текстових файлів

Ця програма дозволяє шифрувати та дешифрувати текстові файли, які містять в собі літери латинського алфавіту, зберігаючи результати у нові файли в тій самій директорії, 
де знаходиться оригінальний файл. Шифрування базується на циклічному зсуві символів.


Функціонал

Читання текстового файлу.
Шифрування вмісту файлу.
Запис зашифрованого тексту у новий файл.
Дешифрування зашифрованого тексту.
Автоматична генерація назв файлів із мітками [ENCRYPTED] або [DECRYPTED].


Як запустити програму

1. Запуск з терміналу:
java -jar "Абсолютний шлях до файлу jar" ENCRYPT "Шлях до текстового файлу/Назва текстового файлу.txt" 20
2. Запуск через середовище розробки:
Після запуску програма надасть інструкцію


Інструкція з використання

Запустіть програму.
Вкажіть шлях до текстового файлу.
 Якщо шлях містить пробіли або спеціальні символи, обгорніть його у лапки.
Виберіть опцію шифрування або дешифрування.
Зашифрований/дешифрований файл буде створено у тій самій директорії з відповідною міткою.
